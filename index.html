ToolBox

HTML 5

Em particular, HTML5 adiciona várias novas funções sintáticas, assim como a integração de conteúdos SVG que substituem o uso de tags object genéricas. Outros novos elementos, como section, article, header e nav, são projetados para enriquecer o conteúdo semântico dos documentos. HTML5 também define com algum detalhe o processamento necessário para que erros de sintaxe de documentos inválidos sejam tratados uniformemente por todos os browsers e outros agentes de usuários em conformidade com o HTML5.

CSS 3

CSS3 é a terceira mais nova versão das famosas Cascading Style Sheets (ou simplesmente CSS ), pela qual se define estilos para um projeto web (página de internet). Com efeitos de transição, imagem, imagem de fundo/background e outros, pode-se criar estilos únicos para seus projetos web, alterando diversos aspectos de design no layout da página. O CSS3 é extremamente capaz de construir animações que impressionam o mais avançado desenvolvedor web, tanto em 2 como em 3 dimensões. Os mais comuns são os efeitos de rotação, movimento e transição.

JavaScript e TypeScript

JavaScript® é uma linguagem leve, interpretada e baseada em objetos com funções de primeira classe, mas usada também em vários outros ambientes sem browser. O JavaScript é uma linguagem baseada em protótipos, multi-paradigma e dinâmica, suportando estilos de orientação a objetos, imperativos e declarativos. Já o TypeScript é um superconjunto sintático estrito de JavaScript que adiciona tipagem estática opcional à linguagem. Tipos fornecem uma maneira de descrever a forma de um objeto, fornecendo melhor documentação e permitindo que o TypeScript valide se seu código está funcionando corretamente.

React e Redux

React JS é uma biblioteca JavaScript para a criação de interfaces de usuário — ou UI ( user interface ). Criado em 2011 pelo time do Facebook, surgiu com o objetivo de otimizar a atualização e a sincronização de atividades simultâneas no feed de notícias da rede social, entre eles chat, status, listagem de contatos e outros. Redux é uma biblioteca para armazenamento de estados de aplicações JavaScript, criado por Dan Abramov. Ele nasceu através de uma implementação do Flux.
